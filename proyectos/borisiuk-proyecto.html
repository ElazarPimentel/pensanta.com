Proyecto Borisiuk

Sistema Multi-Dominio para Fábrica de Mochilas

Cliente: Natacha Borysiuk | Fecha: 27 de Octubre 2025

Descripción General del Proyecto

El proyecto Borisiuk consiste en un sistema de sitios web multi-dominio para la fábrica de mochilas, bolsos y cartucheras industriales de Natacha Borysiuk. El sistema está compuesto por 8 sitios web independientes que comparten una arquitectura común y se alimentan de la misma cuenta de Instagram (@fabricamosmochilas) para mostrar productos de forma dinámica.

Cada sitio web está optimizado para palabras clave específicas (SEO), lo que permite captar búsquedas especializadas en Google para diferentes tipos de productos: mochilas, cartucheras, botineros, etc. Todos los sitios están desplegados en infraestructura gratuita (Vercel) y utilizan un servidor centralizado de API de Instagram (AWS EC2) para obtener las publicaciones de forma eficiente y segura.

Tecnología

Frontend: React 19 + Next.js 15 + TypeScript

Hosting: Vercel (Plataforma gratuita)

Backend API: Node.js + Express.js en AWS EC2
